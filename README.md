# 🚗 Movilidad Urbana y Productividad Económica en Latinoamérica

## Descripción

Este proyecto analiza la relación entre indicadores de movilidad urbana y variables económicas en ciudades latinoamericanas. Se integraron datos de tráfico y economía para explorar patrones que permitan comprender el impacto de la congestión vehicular sobre la productividad.

---

## Objetivos

- Analizar indicadores de tráfico urbano.
- Integrar datos económicos y de movilidad.
- Explorar relaciones entre congestión y productividad.
- Identificar patrones mediante análisis exploratorio.
- Generar conclusiones basadas en evidencia.

---

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Dataset

El análisis utiliza dos conjuntos de datos:

- **TomTom Traffic Index**: indicadores de movilidad urbana.
- **OECD City Economy**: indicadores económicos de ciudades.

---

## Metodología

1. Carga de datos.
2. Limpieza y transformación.
3. Integración de datasets.
4. Análisis exploratorio.
5. Visualización.
6. Interpretación de resultados.

---

## Principales Hallazgos

- Se identificaron diferencias significativas en los niveles de congestión entre ciudades.
- La relación entre PIB per cápita y congestión no mostró una correlación lineal fuerte.
- Algunas ciudades mantienen altos niveles de productividad a pesar de registrar elevados índices de tráfico.
